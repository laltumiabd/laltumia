Web API (রেস্টুরেন্টের বয় ও মেনু কার্ড)

"মামা, API (Application Programming Interface) হলো হোটেলের বয়ের মতো। তুই (Client) কিচেনে (Server) ঢুকতে পারবি না। তুই বয়কে বলবি কী চাস, বয় কিচেন থেকে এনে দেবে। এটাই API।"

HTTP Verbs (অর্ডার দেওয়ার নিয়ম)

API-তে আমরা ৪ ভাবে কথা বলি। লাল্টুর ভাষায় বোঝ:

GET (খাবার দেখা)

মেনু কার্ড দেখা বা ডাটা আনা। কোনো পরিবর্তন হবে না।

POST (অর্ডার করা)

নতুন অর্ডার দেওয়া বা নতুন ডাটা তৈরি করা।

PUT (অর্ডার চেঞ্জ)

বিরিয়ানি বাদ দিয়ে খিচুড়ি নেওয়া। মানে ডাটা আপডেট করা।

DELETE (ক্যান্সেল)

অর্ডার বাতিল করা বা ডাটা ডিলিট করা।

C# Controller Example:
[HttpGet]
public IActionResult GetMenu() 
{
    return Ok("Kacchi, Tehari, Borhani");
}

[HttpPost]
public IActionResult PlaceOrder(string food) 
{
    return Ok(food + " Order Confirm Mama!");
}
