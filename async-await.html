Async & Await (বিরিয়ানি অর্ডার)

"রেস্টুরেন্টে গিয়ে বিরিয়ানি অর্ডার দিয়ে কি কাউন্টারে দাঁড়িয়ে থাকিস? না! তুই টোকেন নিয়ে টেবিলে এসে বসে মোবাইল চাপিস। বিরিয়ানি হলে ওয়েটার ডাক দেয়। এটাই হলো Asynchronous Programming."

Synchronous vs Asynchronous

  • Synchronous (Block): লাইনে দাঁড়িয়ে থাকা। যতক্ষণ কাজ না হচ্ছে, অন্য কিচ্ছু করতে পারবি না।
  • Asynchronous (Non-Block): কাজটা ব্যাকগ্রাউন্ডে চলতে থাকবে, মেইন থ্রেড ফ্রি থাকবে।
Code Example:
// মামা, এখানে 'await' মানে অপেক্ষা করা, কিন্তু ব্লক করা না।
public async Task<string> OrderBiryaniAsync()
{
    Console.WriteLine("Order Pouchaise...");
    
    // বিরিয়ানি হতে ৫ সেকেন্ড সময় লাগবে (Fake Delay)
    await Task.Delay(5000); 
    
    return "Ei nen mama, gorom Biryani!";
}
