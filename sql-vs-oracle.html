ডাটাবেস ম্যানেজমেন্ট (গোডাউন লজিস্টিকস)

"ডাটাবেস হলো তোর মালের গোডাউন। SQL Server হলো মাইক্রোসফটের তৈরি এসি লাগানো গোডাউন, আর Oracle হলো বড়লোকদের বিশাল গোডাউন। দুইজনেই মাল রাখে, কিন্তু খরচ আর কায়দা আলাদা।"
বিষয় SQL Server Oracle
সহজ ভাষায় উইন্ডোজের আপন ভাই। সহজে সেটআপ করা যায়। একটু জটিল, লিনাক্সে ভালো চলে। বড় এন্টারপ্রাইজের জন্য।
ভাষা T-SQL (Transact-SQL) PL/SQL (Procedural Language)
লাল্টুর রায় .NET ডেভেলপার হলে চোখ বন্ধ করে SQL Server ব্যবহার কর মামা। ব্যাংক বা বিশাল কোম্পানিতে গেলে Oracle লাগবে।
