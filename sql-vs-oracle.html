ডাটাবেস ম্যানেজমেন্ট (গোডাউন লজিস্টিকস)

(SQL Server বনাম Oracle এর লড়াই)

চিত্র ১: SQL Server - উইন্ডোজের আপন ভাই (সহজ ও সুন্দর) 💙
চিত্র ২: Oracle - বড়লোকদের বিশাল গোডাউন (খরচে ও পাওয়ারে সেরা) ❤️
"ডাটাবেস হলো তোর মালের গোডাউন। SQL Server হলো মাইক্রোসফটের তৈরি এসি লাগানো গোডাউন, আর Oracle হলো বড়লোকদের বিশাল গোডাউন। দুইজনেই মাল রাখে, কিন্তু খরচ আর কায়দা আলাদা।"
বিষয় SQL Server (Microsoft) Oracle (Oracle Corp)
সহজ ভাষায় উইন্ডোজের আপন ভাই। সেটআপ করা পানির মতো সহজ। GUI খুব ইউজার ফ্রেন্ডলি। একটু জটিল, লিনাক্সে ভালো চলে। বড় বড় ব্যাংক আর মাল্টিন্যাশনাল কোম্পানিতে বেশি চলে।
ভাষা (Language) T-SQL (Transact-SQL) PL/SQL (Procedural Language/SQL)
খরচ (Cost) লাইসেন্স ফি আছে, তবে ছোট কোম্পানির জন্য ফ্রি ভার্সন (Express) আছে। বিশাল খরচ! লাইসেন্স ফি আকাশছোঁয়া। তবে এন্টারপ্রাইজ লেভেলে সেরা।
লাল্টুর রায় .NET ডেভেলপার হলে চোখ বন্ধ করে SQL Server ব্যবহার কর মামা। জীবন সুন্দর হবে। ব্যাংক বা বিশাল টেলিকম কোম্পানিতে গেলে Oracle লাগবে। তখন শিখে নিস।
