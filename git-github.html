Git & GitHub (লাল্টুর টাইম মেশিন)

"কোড লিখতে লিখতে ভুল করে সব ডিলিট করে দিলি? টেনশন নাই! Git হলো টাইম মেশিন। এক ক্লিকে অতীতে ফিরে যাবি। আর GitHub হলো তোর কোডের ক্লাউড অ্যালবাম।"

Common Git Commands (লাল্টুর শর্টকাট):

  • git init: টাইম মেশিন চালু করা।
  • git add .: মালপত্র বস্তায় ভরা (Staging)।
  • git commit -m "msg": বস্তায় সিল মেরে ডেট বসানো (Save Point)।
  • git push: বস্তাটা আকাশে (GitHub) পাঠিয়ে দেওয়া।
Pro Tip: মামা, মেইন ব্রাঞ্চে (Main Branch) সরাসরি পুশ করবি না। জুনিয়র ডেভেলপারদের এটা মহাপাপ! আলাদা ব্রাঞ্চ খুলে কাজ করবি।
