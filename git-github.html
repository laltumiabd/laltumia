Git & GitHub (লাল্টুর টাইম মেশিন)

(ভুল করে কোড ডিলিট হলেও টেনশন নাই!)

চিত্র ১: Git - লাল্টুর কোড রিকভার করার টাইম মেশিন ⏳
চিত্র ২: GitHub - বস্তাভর্তি কোড আকাশে (Cloud) পাঠিয়ে দেওয়া ☁️
"কোড লিখতে লিখতে ভুল করে সব ডিলিট করে দিলি? টেনশন নাই! Git হলো টাইম মেশিন। এক ক্লিকে অতীতে ফিরে যাবি। আর GitHub হলো তোর কোডের ক্লাউড অ্যালবাম। পিসি নষ্ট হলেও কোড আকাশে সেইফ থাকবে।"

⚙️ Common Git Commands (লাল্টুর শর্টকাট):

git init 🚀

টাইম মেশিন চালু করা। ফোল্ডারকে গিট-এর আওতায় আনা হলো।

git add . 🎒

মালপত্র বস্তায় ভরা (Staging)। যা যা চেঞ্জ করেছিস সব রেডি করা।

git commit -m "msg" 🏷️

বস্তায় সিল মেরে ডেট বসানো (Save Point)। যাতে পরে চিনতে পারিস কী কাজ হয়েছিল।

git push ☁️

বস্তাটা আকাশে (GitHub) পাঠিয়ে দেওয়া। এখন তোর পিসি পুড়লেও কোড সেইফ!

🛑
Pro Tip:
মামা, মেইন ব্রাঞ্চে (Main Branch) সরাসরি পুশ করবি না। জুনিয়র ডেভেলপারদের এটা মহাপাপ! আলাদা ব্রাঞ্চ (Branch) খুলে কাজ করবি।
