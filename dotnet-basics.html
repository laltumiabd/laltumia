.NET ইকোসিস্টেম ও CLR

(লাল্টুর ম্যাজিক মার্কেট)

চিত্র ১: লাল্টুর প্রযুক্তির হাট - যা চাবি তাই পাবি!
"শোন মামা, .NET হলো একটা বিশাল বড় বাজারের মতো। তুই যা চাস—ওয়েব, মোবাইল, ডেস্কটপ—সব বানানোর মালমশলা এখানে পাবি। আর এই বাজারের ম্যানেজার হলো CLR (Common Language Runtime)।"

CLR আসলে কী?

তুই যখন C# এ কোড লিখিস, কম্পিউটার সেটা বোঝে না। কম্পিউটার বোঝে 0 আর 1। মাঝখানে একজন অনুবাদক (Translator) দরকার। এই অনুবাদকই হলো CLR।

CLR প্রসেস ডায়াগ্রাম

The Process (লাল্টুর প্রসেস):
  • ১
    তুই লিখলি C# Code (কাঁচা মাল)।
  • ২
    কম্পাইল করলে সেটা হলো IL (Intermediate Language) - এটা আধা-পাকা রান্না।
  • ৩
    রান করার সময় CLR এর JIT (Just-In-Time) কম্পাইলার সেটাকে মেশিন কোডে (0101) কনভার্ট করে।
👨‍💻 C# Code Example 
using System;

namespace LaltuDiary 
{
    class Program 
    {
        static void Main() 
        {
            // তুই লিখলি:
            Console.WriteLine("Hello Mama!");

            // CLR ব্যাকগ্রাউন্ডে সেটাকে প্রসেস করে 
            // মেমোরি ম্যানেজ করে আউটপুট দেখাবে।
        }
    }
}
