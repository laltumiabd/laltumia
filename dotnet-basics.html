.NET ইকোসিস্টেম ও CLR (লাল্টুর ম্যাজিক মার্কেট)

"শোন মামা, .NET হলো একটা বিশাল বড় বাজারের মতো। তুই যা চাস—ওয়েব, মোবাইল, ডেস্কটপ—সব বানানোর মালমশলা এখানে পাবি। আর এই বাজারের ম্যানেজার হলো CLR (Common Language Runtime)।"

CLR আসলে কী?

তুই যখন C# এ কোড লিখিস, কম্পিউটার সেটা বোঝে না। কম্পিউটার বোঝে 0 আর 1। মাঝখানে একজন অনুবাদক (Translator) দরকার। এই অনুবাদকই হলো CLR।

The Process:
  1. তুই লিখলি C# Code.
  2. কম্পাইল করলে সেটা হলো IL (Intermediate Language) - এটা আধা-পাকা রান্না।
  3. রান করার সময় CLR এর JIT (Just-In-Time) কম্পাইলার সেটাকে মেশিন কোডে (Machine Code) কনভার্ট করে।
Code Example: 
using System;
// তুই লিখলি:
Console.WriteLine("Hello Mama!");

// CLR ব্যাকগ্রাউন্ডে সেটাকে প্রসেস করে 
// মেমোরি ম্যানেজ করে আউটপুট দেখাবে।
